# Personal Keymap for YMDK Melody 96

## Keymap Changes

- RGB Controls:
  - Toggle On/Off: `Fn + Tab`
  - Change Mode: `Fn + Q`
  - Increase/Decrease Hue: `Fn + W / Fn + E`
  - Increase/Decrease Saturation: `Fn + R / Fn + T`
  - Increase/Decrease Brightness: `Fn + Y / Fn + U`
- "£" Sign: `Fn + 3`
  - Though I'm unable to get it to work... maybe there's something I missed.
- Toggle NKRO: `Fn + N`
- Menu: `Fn + Alt`
- Print Screen: `Fn + Insert`
- Scroll Lock: `Fn + Delete`
- Pause: `Fn + Home`
- Increase/Decrease Volume: `Fn + Up/Down`
- Increase/Decrease Display Brightness: `Fn + Right/Left`

## Additions to `rules.mk`

- Enable LTO
- Use symmetric, deferred, per-key debounce algorithm
